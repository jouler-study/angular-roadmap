## Run Locally

Clone the project

```bash
  git clone https://github.com/joulert02/angular-roadmap
```

Go to the project directory

```bash
  cd angular-roadmap/03-countries-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  ng serve || npm start
```
